# Document Feed Embedder

![CI/CD](https://github.com/eldius/document-feeder/actions/workflows/ci.yml/badge.svg)

A powerful CLI tool and Proof of Concept (POC) for managing knowledge bases from RSS and OPDS feeds using Retrieval-Augmented Generation (RAG). It leverages Ollama for local LLM processing and embeddings, and `chromem-go` for vector storage.

## Key Features

- **Feed Ingestion**: Support for RSS and OPDS feeds.
- **Vector Database**: Utilizes `chromem-go` for persistent local vector storage.
- **RAG Implementation**: Ask questions to your stored content and get answers generated by local LLMs.
- **Interactive TUI**: Built with Bubble Tea and Lip Gloss for a polished terminal experience.
- **Model Management**: List and configure available Ollama models directly from the CLI.
- **Benchmarking**: Compare different LLM models for performance and quality.
- **Flexible Configuration**: Highly configurable via `config.yaml`.
- **Telemetry**: Built-in support for OpenTelemetry (traces, metrics, logs).

## Prerequisites

- **Go**: 1.26.0 or higher.
- **Ollama**: Must be running locally or accessible via network.
- **Embedding Model**: `nomic-embed-text` (default) or any other supported by Ollama.
- **Generation Model**: `llama3:8b` (default) or any other supported by Ollama.

## Setup & Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/eldius/document-feeder.git
    cd document-feeder
    ```

2.  **Configure the application**:
    Edit `config.yaml` to set your Ollama endpoint and preferred models.
    ```yaml
    ollama:
      endpoint: http://localhost:11434
      embedding:
        model: nomic-embed-text
      generation:
        model: llama3:8b
    ```

3.  **Build the CLI**:
    ```bash
    go build -o document-feed-embedder ./cmd/cli
    go build -o benchmarker ./cmd/benchmarker
    ```

    Alternatively, use the `Makefile`:
    ```bash
    make release
    ```

## Usage

### Managing Feeds

- **Add a feed**:
  ```bash
  ./document-feed-embedder feed add --feed "https://example.com/rss.xml"
  ```
- **List feeds**:
  ```bash
  ./document-feed-embedder feed list
  ```
- **Refresh feeds (Interactive)**:
  ```bash
  ./document-feed-embedder feed refresh -i
  ```
- **Search feeds**:
  ```bash
  ./document-feed-embedder feed search "Golang concurrency"
  ```

### Knowledge Interaction (RAG)

- **Ask a question**:
  ```bash
  ./document-feed-embedder ask "What are the latest trends in Go?"
  ```
- **Output answer to file**:
  ```bash
  ./document-feed-embedder ask "Explain RAG" --output-file answer.txt
  ```

### Model Management

- **List available models**:
  ```bash
  ./document-feed-embedder models
  ```

### Benchmarking

- **Run benchmark**:
  ```bash
  ./benchmarker --model tinyllama:latest --model deepseek-r1:7b
  ```
